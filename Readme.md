# Predictor Mundial FIFA 2026

Aplicación web desarrollada con Flask para:

- Visualizar grupos del Mundial 2026
- Consultar fixture de partidos
- Explorar equipos participantes
- Generar predicciones de resultados
- Interactuar con un chatbot basado en IA

## Tecnologías

- Python
- Flask
- HTML
- CSS
- GitHub

## Módulos

- Inicio
- Grupos
- Fixture
- Equipos
- Predicciones
- Chatbot IA
